The one theme for [bisheng](https://github.com/benjycui/bisheng).

## Features

* Load Markdown data lazily.
* Server-side render for SEO.

## Usage

Installation:

```bash
npm i --save bisheng-theme-one
```

Configuration in `bisheng.config.js`:

```js
{
  theme: 'bisheng-theme-one',
  ...
}
```
